# Meetbased project based on moodle project
**Main moodle project description:** Moodle is a free and open-source learning management system written in PHP and distributed under the GNU General Public License. Moodle is used for blended learning, distance education, flipped classroom and other online learning projects in schools, universities, workplaces and other sectors.
Moodle is used to create custom websites with online courses and allows for community-sourced plugins.
